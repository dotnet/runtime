<h3>Join Microsoft / .Net Foundation organizations</h3>

1. Link Microsoft account with the Github account and join Microsoft GitHub organization:
    + Go to https://opensourcehub.microsoft.com/link/ and https://repos.opensource.microsoft.com/ to link.
    + Go to https://repos.opensource.microsoft.com/Microsoft to join Microsoft org.
    + Go to  https://repos.opensource.microsoft.com/DotNet to join DotNet org.
    + Join this group as well https://repos.opensource.microsoft.com/dotnet/teams/mlnet if you want to do PR via codeflow.
2. Publicly associate yourself with Microsoft:
    + Go to https://github.com/orgs/Microsoft/people 
    + Search for your name in the list. 
    + Dropdown list next to your name -- change visibility from private to public. 
    + People will now see your organization affiliation (you see this on your own account whether your association is public or private). 
3. Publicly associate yourself with the .NET Foundation: 
    + Go to https://github.com/orgs/dotnet/people 
    + Search for your name in the list. 
    + Dropdown list next to your name -- change visibility from private to public.
